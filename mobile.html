My Projects

Embracing Olin's do-learn mentality, I've worked on a number of really meaningful and educational projects from Android apps to mechnical hedgehog toys.

Hopper Project

A simple project that served as an introduction to the design process and Olin's do-learn mentality

Transporter Project

An bio-inspired interactive game for fourth graders design for Olin's Design Nature class that used the design process

Hashtag Gloves

Gloves that allowed users to tweet by tapping their fingers together. It won 3rd place at HackHolyoke 2014

Tracky Bird

Interactive twist on the popular game flappy bird using Python's OpenCV module to require players to flap their wings

Chordly

Personal project that helped musicians write songs by helping to match guitar chords.

Kyzr

Virtual torch passing app for Android that allowed users to pass "torches" using NFC and track their progress online
